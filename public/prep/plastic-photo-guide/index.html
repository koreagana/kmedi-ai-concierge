1. 拍摄基本原则
촬영 기본 원칙
• 素颜拍摄，不可化妆（包括遮瑕膏、BB霜）。
반드시 맨 얼굴로 촬영하세요. 컨실러, BB크림 포함 화장 금지.
• 在自然光线下拍摄（靠近窗边），避免使用闪光灯或美颜滤镜。
자연광 아래(창가)에서 촬영하세요. 플래시와 뷰티 필터 사용 금지.
• 将头发完全向后拢起，充分露出耳朵和发际线。
머리카락을 완전히 뒤로 묶어 귀와 헤어라인이 충분히 보이도록 하세요.
• 摘除眼镜、耳环等饰品。
안경, 귀걸이 등 액세서리를 모두 제거하세요.
• 摄像头与面部保持同一水平线，不要仰拍或俯拍。
카메라를 얼굴과 같은 높이에 맞추세요. 올려 찍거나 내려 찍지 마세요.

3. 各部位追加拍摄
부위별 추가 촬영
👁️ 眼部手术 / 눈 수술 (쌍꺼풀·트임·눈밑)
• 正面：自然睁眼 / 用力睁眼 / 闭眼 各一张
정면: 자연스럽게 눈 뜬 상태 / 힘껏 눈 뜬 상태 / 눈 감은 상태 각 1장
• 眼睛特写（正面）：清晰拍摄双眼皮线条、内眼角、下眼睑状态
눈 클로즈업 (정면): 쌍꺼풀 라인, 내안각, 하안검 상태가 선명하게 찍히도록
👃 鼻部手术 / 코 수술
• 正面 / 左右侧面 / 仰面（鼻底） / 俯视（鼻梁） 各一张
정면 / 좌우 측면 / 앙각 (코 밑) / 코 위에서 내려다본 사진 각 1장
🏛️ 面部轮廓 / 안면윤곽 (광대·사각턱·턱끝)
• 正面 / 左右侧面 / 左右45° / 仰面 追加拍摄，充分展示骨骼轮廓

• 除正面、左右45°基础拍摄外，请额外加拍您特别关注部位的特写。

# shop-preview/
쇼핑몰 페이지 HTML 초안 보관 폴더 (미오픈).
쇼핑몰 오픈 시 이 파일들을 참고하여 React 컴포넌트로 변환합니다.

| 파일 | 설명 |
|------|------|
| shop-main.html | 쇼핑몰 메인 목록 페이지 |
| shop-detail.html | 상품 상세 페이지 (범용) |
| shop-order.html | 주문/결제 페이지 |
| pharmicell-cream.html | 플레이 셀 판테놀 닥터 크림 상세 |
| pharmicell-ampoule.html | 플레이 셀 판테놀 힐러 앰플 상세 |
